# 🎓 Student Burnout Prediction using Machine Learning

## 📌 Project Overview

Student Burnout Prediction is a Machine Learning-based web application that predicts the burnout risk of students based on academic and lifestyle factors.

The application uses a Random Forest Classifier trained on student data and provides personalized recommendations.

---

## 🚀 Features

- Predict student burnout risk
- Three prediction levels:
  - Low
  - Medium
  - High
- Prediction confidence
- Personalized recommendations
- Feature importance visualization
- Interactive Streamlit web application
